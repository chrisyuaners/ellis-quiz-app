# mod-3-project
Flatiron School - Ellis Quiz App

A single page application that lets users prepare for the U.S. neutralization test.

Make sure you are running the client and backend folders separaetly.

## Client:

Install dependencies -
### `yarn install`
or
### `npm install`

Run server -
### `yarn start`
or
### `npm start`

## Backend:

Install dependencies -

### `bundle install`

Run server -

### `rails s`
